SQLite3 to MySQL¶

A simple Python tool to transfer data from SQLite 3 to MySQL

Installation¶

pip install sqlite3-to-mysql

Basic Usage¶

sqlite3mysql -f path/to/foo.sqlite -d foo_db -u foo_user -p
